# Mini Java Game 

A simple Rock-Paper-Scissors game built in Java where the user plays against the computer.

## Features

* User input with `Scanner`
* Random computer choice
* Win, lose, and draw conditions
* Input validation

## How to Run

1. Compile and run `Main.java`
2. Enter:

   * `0` for Rock
   * `1` for Paper
   * `2` for Scissors

## Tech Stack

* Java

This project was created to practice Java fundamentals such as arrays, conditionals, user input handling, and random number generation.
